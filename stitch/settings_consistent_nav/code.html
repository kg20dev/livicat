Connect to OBS

Follow these steps to integrate your custom chat theme directly into your broadcast environment.

1

Open OBS Studio

Navigate to your 'Sources' panel and click the + icon.

2

Add Browser Source

Select Browser from the list and name it "Stream Chat".

3

Paste Custom CSS

Paste the generated code from the right panel into the Custom CSS box.

4

Apply & Preview

Click OK. Your chat will now appear with the StreamStyle theme applied.
